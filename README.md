# 100PrintsWithMe Verification App

Official Android verification application for the **100PrintsWithMe** platform.

The app allows organizers, schools, NGOs, companies, and event staff to instantly verify QR-enabled documents generated using 100PrintsWithMe.

---

## Features

- 📷 Fast QR Code Scanner
- ⚡ Instant Document Verification
- 🎫 Verify Event Passes
- 🪪 Verify ID Cards
- 🏆 Verify Certificates
- 💳 Verify Membership Cards
- 📱 Native Android Application
- 🔒 Secure verification through the 100PrintsWithMe API

---

## Supported Documents

The app can verify documents generated by the 100PrintsWithMe platform, including:

- Certificates
- Student ID Cards
- Employee ID Cards
- Event Badges
- Event Tickets
- Membership Cards
- Volunteer Cards
- Visitor Passes
- Admission Cards

## How it Works

```
Open Scanner

      ▼

      ▼

      │
      ▼
100PrintsWithMe API
      │

      │
      ▼
Display Verification Status
```

Clone the repository.

```bash

```

Open the project using Android Studio.

Run on any Android device with Camera permission enabled.

## About 100PrintsWithMe

[100PrintsWithMe](https://100printswith.me) is a document generation platform that enables organizations to design once and generate personalized certificates, ID cards, badges, tickets, memberships, and other printable documents using dynamic data.

The Verification App is the official Android companion application for validating those documents in real time.
